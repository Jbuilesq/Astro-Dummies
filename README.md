### Astrodummies (Astronomia para dummies)

## Idea
Astronomia para dummies es una pagina para aprender y entender conceptos basicos de astronomia, visualmente atractiva para motivar a las personas a prender sobre este tema

## Problema a resolver:
El "problema" a resolver es motivar a la gente para estidiar astronomia y que esten mas motivados a estudiar e investigar sobre ese tema, mas que un problema a resolver es crear una facilidad para estudiar algo tan fundamental como la astronomia

## Publico objetivo:
Jovenes y adultos entre los 14 y 26 pero la idea es no limitarse a ese publico


## Funcionalidades clave:
- Debe ser intuitiva
- Informacion detallada sobre astronomia
- Debe ser atractiva para el usuario

## Funcionalidades:
- La pagina debe ser responsiva
- Debe ser intuitiva
- Informacion detallada sobre astronomia
- La pagina debe contar con animaciones 
- La pagina debe tener una area de un formulario de contacto
- Imagenes explicativas
- La pagina debe tener elemtos interectivos 
- Debe ser atractiva para el usuario
- La pagina debe tener un logo en la parte de la pestana

## Historias de usuario:
Como usuario quiero tener secciones sobre cada tema.
Como usuario me gustaria tener imagenes o videos de astronomia.
Como usuario quiero tener diferentes niveles de informacion por si mi nivel es mas bajo.

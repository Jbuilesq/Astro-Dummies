### Astrodummies (Astronomia para dummies)

## Idea
Astronomia para dummies es una pagina para aprender y entender conceptos basicos de astronomia, visualmente atractiva para motivar a las personas a prender sobre este tema

## Problema a resolver:
El "problema" a resolver es motivar a la gente para estidiar astronomia y que esten mas motivados a estudiar e investigar sobre ese tema, mas que un problema a resolver es crear una facilidad para estudiar algo tan fundamental como la astronomia


## Funcionalidades clave:
- Debe ser intuitiva
- Informacion detallada sobre astronomia
- Debe ser atractiva para el usuario

## Funcionalidades:
- La pagina debe ser responsiva
- Debe ser intuitiva
